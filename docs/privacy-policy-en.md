## 2. Data We Collect

### 2.1 Account Data
- Username
- Email address
- Password hash (the original password is never stored)
- Organization name

### 2.2 Business Data
Data you enter while using the Application:
- Contacts: name, phone, email, company, notes
- Deals: title, amount, stage, linked contact
- Tasks: title, due date, status, priority
- Calendar events: title, time, location, attendees

### 2.3 Geolocation (optional)
Geolocation is requested only when creating a field visit or meeting, and only with your explicit consent. Location data is not tracked continuously.

### 2.4 Technical Data
We do **not collect** and do **not transmit** device information, operating system data, advertising identifiers, or in-app behavioral analytics. No third-party analytics SDKs (Firebase, Amplitude, or similar) are used.
